# Compact Communication

Compact does not mean vague. Preserve:

- source;
- fact/deduction distinction;
- exceptions;
- unresolved decisions;
- verification state;
- next owner.

Remove:

- repeated task restatement;
- long narrative chronology;
- copied source blocks;
- praise and conversational filler;
- obvious explanations of protocol fields;
- unchanged previous records.

Use IDs so later messages can say `DEC|D3|...|F2,F7` instead of copying the evidence.
