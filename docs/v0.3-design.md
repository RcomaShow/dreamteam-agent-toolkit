# DreamTeam 0.3 — Executable Cost-Proof Orchestration

Version 0.3 ships a dependency-free runtime both at repository root and inside the installable Claude Code plugin. The adapter sync copies the canonical runtime into `plugin/lib/dreamteam`, generates agents and references, and CI rejects drift.

Runtime configuration is strict and reproducible: pricing date is explicit, unknown properties fail, source-content telemetry is forbidden, and independent writer review plus anchor validation are mandatory.

The routing model keeps a Sonnet 5-only baseline, accounts for every active Lean/Frontier tier, gates Batch on four conditions, and exposes selected and rejected costs. SQLite uses exact byte-span union/intersection, `BEGIN IMMEDIATE` reservations, release/reconciliation, decision-critical permits, and metadata-only invalidations.

Plugin hooks cover Pre/Post tool use and subagent lifecycle. Advisory mode warns; strict benchmark mode denies unaccountable Bash reads, nested Agent dispatch, missing budget reservations, and rereads above policy without a permit.

The artifact ZIP is built deterministically and smoke-tested after extraction with an empty `PYTHONPATH`. No provider executor or hidden paid inference is bundled.
