# Gemini CLI Adapter

Planned adapter requirements:

- preserve DT-C1 authority and execution routes;
- map discovery, execution, and verification worker charters;
- render DCP/2 and parse CHP/2;
- maintain TLP/2 state and explicit code ownership;
- keep C3 decisions with the selected high-capability orchestrator.

This directory is an extension contract, not a working adapter yet.
