# Changelog

All notable changes follow semantic versioning.

## [0.1.0] - 2026-07-16

### Added

- Platform-independent DreamTeam core protocols.
- Token-aware routing and escalation policies.
- Compact delegation, handoff, and task-ledger formats.
- Economy, balanced, and quality profiles.
- Six generic worker specifications.
- Claude Code marketplace and `dreamteam` plugin adapter.
- `/dreamteam:run`, `/dreamteam:review`, and `/dreamteam:measure`.
- Structural validator, adapter synchronization tool, release builder, and tests.
- Extension contracts for Codex, Gemini CLI, and generic runtimes.
